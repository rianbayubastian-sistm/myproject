# VGOS v1.0 MVP

Visual Generation Operating System untuk mengelola project visual AI dari input, analyze, knowledge detection, scene editor, compiler, QC, production, research, hingga report.

## Freeze Contract

Konsep menu dan workflow dibekukan sesuai VGOS v1.0:

- Dashboard
- Projects
- Analyze
- Scene Editor
- Knowledge
- QC
- Production
- Research
- Reports
- Settings

Milestone 1 fokus pada fondasi:

- Dashboard
- Projects
- Analyze gambar dan teks
- Scene Editor
- Compiler sederhana

## Run

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Version

VGOS `0.1.0`

Status: Milestone 1 foundation.
